# Repertoire Trainer

A lightweight Progressive Web App (PWA) for managing and practicing chess opening repertoires. 

## Features
- Complete offline capability (PWA architecture)
- Practice and Edit modes
- Board customization (themes)
- Zero build tools required - just static files

## Architecture
Built with pure Vanilla HTML, CSS, and modern JavaScript (Native ES Modules). No bundlers or package managers are required.
